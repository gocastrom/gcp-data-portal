# Architecture (MVP)
- Frontend: React (Vite)
- Backend: FastAPI
- Catalog Search: Dataplex Universal Catalog / Data Catalog search
- Preview: BigQuery (optional)
- Access Requests: SQLite (MVP) -> Firestore (prod)
- Approvals: IT Owner + Data Owner + Data Steward
- Provisioning: IAM grant stub (MVP) -> dataset/table IAM in prod
